# Employee Attrition Prediction

## 🎯 Overview
This project focuses on **Employee Attrition Prediction**, a critical HR analytics application. The goal was to build a machine learning system that predicts whether an employee is likely to leave the company based on factors like job satisfaction, salary, work-life balance, years at the company, and performance ratings. By analyzing this data, we provide HR departments with actionable insights to improve retention strategies.

## 📊 Project Workflow (7 Tasks)
1. **Data Loading & Exploration:** Initial data profiling and attrition rate analysis.
2. **Data Cleaning & Preprocessing:** Handling null values, feature encoding, and normalization.
3. **Exploratory Data Analysis (EDA):** Identifying patterns via visual data inspection.
4. **Model Building:** Training and comparing Logistic Regression, Random Forest, and Gradient Boosting classifiers.
5. **Model Evaluation:** Performance assessment using Precision, Recall, F1-Score, and ROC-AUC.
6. **Visualization:** Creating professional charts (Confusion Matrix, Feature Importance, etc.).
7. **HR Recommendations:** Translating technical model outputs into strategic business advice.
